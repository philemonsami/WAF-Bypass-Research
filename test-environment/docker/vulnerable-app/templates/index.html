WAF Validation Framework

A purposefully insecure backend designed to evaluate the rule sets, normalization behaviors, and bypass mitigation strategies of Web Application Firewalls in real-time.

SQL Injection

GET /sqli?username=...

Tests database parsing impedance. Appends raw string inputs to a SQLite backend, allowing testing of inline comments (`/**/`), hex encoding, and whitespace manipulation.

Cross-Site Scripting

GET /xss?name=...

Evaluates frontend sanitation regex. Try injecting obscure HTML5 event handlers (`onpointerenter`) or utilizing double URL encoding to bypass filter layers.

Command Injection

POST /cmd?ip=...

Triggers OS-level commands. Useful for checking if WAF rules accurately identify common shell separators like `;`, `&&`, or parameter passing mechanisms.
▤

Local File Inclusion

GET /lfi?file=...

Checks directory traversal normalization. Use self-referential directories (`/././`) or over-slashing (`////`) to check WAF path resolving limits.

Server-Side Request

GET /ssrf?url=...

Analyze SSRF metadata and protocol handlers. Attempt to fetch internal resources (e.g., `127.0.0.1` or `169.254.169.254`) utilizing varied input structures.
⟳

Eval Injection

GET /eval?calc=...

Test deeply nested payload execution via Python `eval()`. Excellent for bypassing signature engines relying strictly on language-specific heuristics.

Testing Methodology Details

Impedance Mismatch

When the WAF decodes a payload differently than this backend application, bypasses occur. This environment relies heavily on Python and Flask’s specific URL parsing logic. Test if the WAF catches multi-layer encoding.

Request Smuggling

By leveraging an intermediary proxy (like a WAF), manipulate the `Content-Length` and `Transfer-Encoding` headers in your HTTP requests. See if you can desync the pipeline leading up to this server.

Parameter Pollution

Send multiple instances of the same parameter (e.g., `?username=1&username=2`). Flask natively grabs the first parameter by default when using `request.args.get()`, while many WAFs inspect the last.

Obfuscation Tolerance

Automate testing of this endpoint using the Python scripts provided in the `/experiments/` directory to methodically insert junk bytes, inline comments, and syntax variations to exhaust regex processing engines.
